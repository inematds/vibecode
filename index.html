O que é Vibe Coding?

Em fevereiro de 2025, o co-fundador da OpenAI Andrej Karpathy cunhou o termo: desenvolver software descrevendo o que você quer em linguagem natural para uma IA, que gera o código automaticamente.

💬

Você descreve

Em português mesmo. "Quero um dashboard com os dados dessa planilha, com filtros por mês."

🤖

A IA constrói

O código é gerado automaticamente. Você testa o resultado, não cada linha de código.

🚀

Você itera

Pede ajustes em linguagem natural até o resultado estar exatamente como você quer.
