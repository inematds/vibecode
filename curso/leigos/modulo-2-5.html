🔗 O que é uma API — A explicação mais simples

API significa "Interface de Programação de Aplicações" — mas esse nome não ajuda ninguém. A analogia que funciona: pense em um garçom de restaurante. Você (o seu app) pede algo ao garçom (API), ele vai à cozinha (outro sistema) e traz o que você pediu. Você nunca entra na cozinha — apenas faz o pedido.

🍽️ A analogia do restaurante

🧑

Você (seu app)

Quer saber o endereço de entrega

🤵

Garçom (API)

Leva o pedido para o Google Maps

🗺️

Cozinha (Google Maps)

Retorna o endereço formatado

📊 APIs que você já usa sem saber

  • • Quando seu app "entra com Google" — API de autenticação do Google
  • • Quando paga no iFood — API do Stripe/Mercado Pago processando o cartão
  • • Quando o Uber mostra sua posição — API do Google Maps
  • • Quando o WhatsApp recebe a mensagem automática — API do WhatsApp Business

💡 O que você precisa saber como não-técnico

Você não precisa entender como as APIs funcionam internamente. Você precisa saber: qual API usar para cada função, como pedir para a ferramenta de vibe coding integrá-la, e como testar se está funcionando.

💳 Pagamentos — Stripe e Mercado Pago

Integrar pagamentos costumava ser o maior pesadelo de um MVP. Hoje, com vibe coding, você descreve o que quer e a ferramenta integra o Stripe ou Mercado Pago em minutos. O segredo está em como você descreve o fluxo de pagamento.

Stripe (recomendado para global)

  • • Aceita cartão de crédito, débito, PIX
  • • Melhor para assinaturas recorrentes
  • • Dashboard completo de análise
  • • Taxa: ~2.9% + R$0.30 por transação
  • • Integração com Lovable via um prompt

Mercado Pago (melhor para Brasil)

  • • PIX, boleto, cartão, QR code
  • • Melhor reconhecimento do consumidor BR
  • • Integração com e-commerce nacional
  • • Taxa variável por volume
  • • Suporte em português

💬 Como pedir integração de pagamento

"Adicione um botão de 'Assinar por R$49/mês' que abre um checkout do Stripe para cobrança recorrente mensal."
"Quando o usuário clicar em Pagar, abra o checkout do Mercado Pago com o valor de R$197 via PIX ou cartão."
"Após pagamento confirmado, libere acesso ao conteúdo premium e envie email de confirmação."

⚠️ Importante: modo de teste

Sempre teste pagamentos no modo sandbox (teste) antes de ativar em produção. Stripe e Mercado Pago fornecem cartões de teste para simular compras sem movimentar dinheiro real. Peça para a IA usar as credenciais de teste primeiro.

📱 WhatsApp e email — Notificações automáticas

Notificações automáticas são o tipo de integração que mais impressiona os usuários e mais economiza tempo do dono do negócio. Um lembrete de agendamento enviado automaticamente pelo WhatsApp pode reduzir no-shows em até 80%.

Email automático com Resend ou SendGrid

O mais simples de integrar. Gratuito para até 100 emails/dia no Resend.

"Quando um usuário se cadastrar, envie um email de boas-vindas com o nome dele."
"24 horas antes do agendamento, envie um email de lembrete para o cliente."

WhatsApp via Twilio ou Z-API

Mais impactante, mas requer aprovação do template pelo WhatsApp Business.

"Quando confirmar agendamento, envie mensagem WhatsApp para o cliente com os dados."
"Integre com Z-API para enviar lembretes de agendamento 1 hora antes via WhatsApp."

💡 Comece com email, depois adicione WhatsApp

Email é mais simples de configurar e não requer aprovação prévia. Valide que o fluxo de notificação funciona com email primeiro. Depois que estiver tudo certo, adicione WhatsApp como canal adicional.

📊 Google Sheets — Planilha como banco de dados

Para MVPs muito simples ou para donos de negócio que já controlam tudo em planilhas, integrar com Google Sheets é uma solução elegante. Você mantém o controle familiar da planilha e adiciona um app bonito na frente.

✓ Quando usar Google Sheets

  • ✓Você ou sua equipe já trabalham com planilhas
  • ✓Volume de dados é pequeno (abaixo de 10K linhas)
  • ✓Você precisa que não-técnicos editem os dados
  • ✓MVP para validar antes de banco real

✗ Quando NÃO usar

  • ✗Mais de 100 usuários simultâneos
  • ✗Dados sensíveis (senhas, cartões)
  • ✗Necessidade de transações rápidas
  • ✗Produto escalável a longo prazo

💬 Como pedir a integração

"Quando um usuário preencher o formulário de contato, salve as informações numa planilha do Google Sheets que eu vou criar. Preciso da API Key do Google Sheets."

⚙️ Zapier/n8n — Quando a IA não consegue

Às vezes a ferramenta de vibe coding não consegue integrar com um sistema específico — especialmente sistemas muito nichos ou legados. Nesses casos, Zapier e n8n são os "tradutores universais" que conectam qualquer coisa com qualquer coisa.

Zapier (sem código, pago)

  • • Interface visual sem código
  • • 6.000+ apps integrados
  • • Gratuito para 100 automações/mês
  • • Melhor para não-técnicos totais

n8n (open source, flexível)

  • • Gratuito para self-hosted
  • • Mais flexível que Zapier
  • • Curva de aprendizado maior
  • • Melhor para quem tem um dev para configurar

💡 O fluxo Zapier para não-técnicos

Quando [TRIGGER acontece no SEU APP] → Zapier faz [AÇÃO em OUTRO APP]

• Novo cadastro → Adiciona na lista do Mailchimp

• Pagamento aprovado → Cria task no Trello

• Formulário enviado → Notifica no Slack

🧠 OpenAI/Claude na sua aplicação — IA dentro do produto

Adicionar IA ao seu produto transforma uma ferramenta comum em algo que parece mágico para o usuário. Você pode integrar a OpenAI (ChatGPT) ou a Anthropic (Claude) diretamente no seu app — e cobrar por isso.

1

Chatbot de suporte automatizado

IA responde perguntas dos clientes usando o conteúdo do seu site ou base de conhecimento. Reduz suporte em até 70%.

"Adicione um chatbot no canto inferior direito que responde perguntas sobre meu produto usando as FAQs da minha base de conhecimento."
2

Geração de conteúdo personalizado

Usuário preenche dados, IA gera texto personalizado. Contratos, emails, descrições de produto, relatórios.

"Quando o usuário preencher os dados do cliente, gere automaticamente um email de proposta personalizado usando IA."
3

Análise e classificação automática

IA lê textos, classifica sentimento, extrai informações ou sugere ações. Perfeito para análise de feedback, triagem de leads.

"Quando um feedback chegar, use IA para classificar como positivo/neutro/negativo e extrair o problema principal."

📊 Custo de integrar IA (OpenAI)

  • • GPT-4o: ~$0.005 por chamada (texto médio) — muito barato para MVPs
  • • Para 1.000 usuários/dia: custo de IA tipicamente abaixo de $50/mês
  • • Você pode cobrar $10-30/mês dos usuários e ter margem

🧪 Testando integrações — Como verificar que está funcionando

Integração parece estar funcionando quando você testa uma vez e funciona. Mas uma integração está realmente funcionando quando você testa cenários de erro: o que acontece quando o serviço externo demora, quando o dado está errado, quando o pagamento é recusado.

Checklist de teste de integração

  • □Fluxo principal funciona?
  • □O que acontece em caso de erro?
  • □Pagamento recusado é tratado?
  • □Email chega na caixa de entrada (não spam)?
  • □WhatsApp chega em menos de 1 minuto?
  • □Dados estão corretos no dashboard?

Ferramentas de monitoramento

  • • Stripe Dashboard: veja todas as transações e falhas
  • • Resend/SendGrid: veja emails entregues e rejeitados
  • • Supabase logs: veja erros de banco de dados
  • • Zapier History: veja automações executadas e com erro

💡 A regra do "e se der errado"

Para cada integração, pergunte: "E se der errado, o usuário fica sabendo?" Se o pagamento falhar silenciosamente, o usuário pode achar que pagou quando não pagou. Sempre peça para a IA tratar os erros com mensagens claras.
